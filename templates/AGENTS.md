## Retrieval rules
1. RULES is the floor and is always loaded first; BOOT is its digest for tight contexts.
2. Route by task via BOOT's table; read the matching canon file *before* generating in a known domain — if no canon, say so and proceed from RULES (degradation, honestly labeled).
3. Owner voice is data with authorship: quote takes verbatim, never paraphrase them into new claims, never fill empty ones.
4. Chunk reads per section; this file's table is the map, not the territory.
