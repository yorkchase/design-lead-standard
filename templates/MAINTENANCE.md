# MAINTENANCE — how the instance grows without drifting *(near-complete template)*

*This governance is structural, not taste — adopt it nearly whole. The organizing idea: autonomy is routed by stakes; agents propose, the owner ratifies; high stakes never self-approve.*

**Ratifier:** [name] · **Retro cadence:** [monthly]

## The two learning channels

| Channel | Governance |
| --- | --- |
| **The owner's voice** — takes, §G rules | Never autonomous, never on demand. Enters only through real-use corrections, drafted in the owner's words, ratified explicitly. Empty stays empty. |
| **Evidence-based doctrine** — patterns, protocols, fixtures | Agents may *propose* changes (including reversals) with cited evidence. Proposals, not commits. |

## The ladder, applied to edits

- **Act + log:** broken links, stale counts, regenerating the sealed boot, writing retro reports. Direct, committed, revertible.
- **Propose:** doctrine changes, new fixtures, structural moves. Ratified before landing.
- **Always ask, never graduates:** anything in §A · anything in the owner's voice · deletions · licensing/visibility changes.

## Cadences

- **Per run — harvest:** every engagement ends with candidates drafted from the owner's corrections; the owner ratifies.
- **Retro ([monthly]):** re-run the smoke brief sealed on the daily model and compare to baseline (**a score drop is the drift alarm**) · linkcheck · regenerate + republish the sealed boot if sources changed · sweep `harvests/INBOX.md` into drafted proposals (mark swept lines, never delete) · list rules uncited since the last retro (the owner decides if they're universal or unnecessary) · write the report; the report never self-ratifies anything.
- **Continuous — calibration:** the owner periodically scores a run finding-by-finding. Their verdicts are the ground truth that keeps "consistent" and "theirs" the same thing.

## Guardrails
Over-capture is a failure mode — the harvest bar is *durable, generalizable, theirs*. One source of truth: corrections made in other tools come home as candidates, never parallel constitutions. Reports never self-ratify.
